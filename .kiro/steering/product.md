# Product Overview

Haramaya University Ethics and Anti-Corruption Club web application.

## Purpose
A platform for promoting ethical conduct, combating corruption, and fostering accountability within the university community through:
- Anonymous reporting system for ethical concerns
- Educational programs and resources
- Event management and news updates
- Election system for club leadership
- Member management and engagement

## Key Features
- Public-facing pages: Home, About, Programs, Achievements, Events, News, Resources
- Anonymous report submission system
- Voting/election system for club positions
- Admin dashboard for content and user management
- Dark/light theme support
- Role-based access control (admin vs regular users)

## Target Users
- University students and staff (general public pages)
- Club members (voting, joining)
- Administrators (content management, report handling, elections)
